# E-Commerce Platform

Полнофункциональный интернет-магазин с административной панелью, реализованный на Laravel и Livewire.  
Включает пользовательскую часть магазина, корзину, оформление заказов, личный кабинет пользователя и расширенную админку для управления контентом и заказами.


---

## Основной функционал

### Пользовательская часть
- Каталог товаров с многоуровневыми категориями
- Фильтрация товаров:
    - по атрибутам
    - по цене
- Сортировка и пагинация
- Поиск по товарам
- Страница товара с подробным описанием
- Хлебные крошки
- Корзина:
    - добавление / удаление товаров
    - изменение количества
    - мини-корзина
- Оформление заказа
- Email-уведомления о заказе
- Регистрация и аутентификация пользователей
- Личный кабинет пользователя:
    - редактирование профиля
    - просмотр списка заказов
    - просмотр деталей заказа
- SEO-оптимизация страниц

---

### Административная панель
- Отдельная админ-зона с middleware
- Dashboard
- Управление категориями:
    - создание
    - редактирование
    - удаление
- Управление товарами:
    - CRUD
    - загрузка изображений
    - визуальный редактор описаний
    - управление ценами и статусами
- Система фильтров:
    - группы фильтров
    - фильтры
    - привязка фильтров к категориям
- Управление заказами:
    - просмотр
    - удаление
- Управление пользователями:
    - создание
    - редактирование
    - удаление

---

## Технологический стек

- **Backend:** PHP 8+, Laravel
- **Frontend:** Blade, Livewire, Bootstrap
- **Database:** MySQL
- **ORM:** Eloquent
- **Authentication:** Laravel Auth
- **Email:** SMTP (Gmail)
- **Environment:** Docker / локальный сервер
- **Version control:** Git

## Установка и запуск

```bash
git clone https://github.com/lomovlad/ecommerce-platform.git
cd ecommerce-platform

composer install
cp .env.example .env
php artisan key:generate

php artisan migrate --seed
php artisan serve
